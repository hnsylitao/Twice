# Scripts
**install**
```js
$ npm install
```
**start**
```js
//run dev
$ npm run start
//run test
$ npm run test
//run prod
$ npm run prod
```
**build**
```js
// build dev
$ npm run build-dev
// build test
$ npm run build-test
// build prod
$ npm run build-prod
```
